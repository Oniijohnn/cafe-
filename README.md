# cafe
Cafe official bot.

## Setup

1. Create a `.env` file in the root directory and add the following:
    ```
    BOT_TOKEN=your-bot-token
    CLIENT_ID=your-client-id
    GUILD_ID=your-guild-id
    ```

2. Install dependencies:
    ```
    npm install
    ```

3. Run the bot:
    ```
    npm start
    ```
